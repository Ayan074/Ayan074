<!-- ━━━━━━━━━━━━━━ SNAKE ANIMATION SETUP GUIDE ━━━━━━━━━━━━━━ -->

<details>
<summary><h3>🐍 How to Enable Contribution Snake</h3></summary>

1. Go to your **`Ayan074`** repository
2. Create `.github/workflows/snake.yml` with this content:

```yaml
name: Generate Snake Animation

on:
  schedule:
    - cron: "0 */12 * * *"  # Runs every 12 hours
  workflow_dispatch:

jobs:
  build:
    runs-on: ubuntu-latest
    steps:
      - name: Generate Snake Game
        uses: Platane/snk@v3
        with:
          github_user_name: Ayan074
          outputs: |
            dist/github-snake.svg
            dist/github-snake-dark.svg?palette=github-dark

      - name: Push to output branch
        uses: crazy-max/ghaction-github-pages@v3.1.0
        with:
          target_branch: output
          build_dir: dist
        env:
          GITHUB_TOKEN: ${{ secrets.GITHUB_TOKEN }}
```

3. Go to **Actions** tab → **Run workflow** manually for the first time
4. The snake animation will auto-update every 12 hours! 🎉

</details>
